## Components

- `ChallengeDetails` вҖ” name, dates, wager, companions
- `ProgressTrail`, `LivesIndicator`
- Primary `Button` вҖ” context-aware CTA
- `ActivityFeed` вҖ” no message input (US-4 omitted)
- `ApprovalWidget` вҖ” companion only on pending proof
- `MediaUploadSheet` вҖ” from Check In
- `ProofViewer` вҖ” companion verify
- Menu вӢ® вҖ” Leave challenge (companion only)

## Challenger actions

- Check In вҶ’ upload sheet
- Settle up вҶ’ wager upload (failed state)

## Companion actions

- Verify вҶ’ proof viewer + Looks good / Try again
- Leave вҶ’ confirm dialog
